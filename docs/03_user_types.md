# Tipos de usuário

## Usuário local

**Quem:** Pessoa que organiza tarefas pessoais ou profissionais no próprio computador.

**Contexto:** Uso individual, offline, sem equipe. Abre o app, escolhe ou continua um quadro, gerencia colunas e cartões ao longo do dia.

**Nível técnico:** Básico a intermediário — espera interface intuitiva estilo Trello, sem configuração de servidor ou conta.

**Necessidades:**

- Ver quadros ou retomar o último quadro imediatamente ao abrir.
- Criar e reorganizar colunas e cartões com fluidez.
- Registrar detalhes nas tarefas (descrição, prioridade, etiquetas, prazo, checklist).
- Confiar que alterações são salvas automaticamente.

**Fora do perfil v1:** Administrador de sistema, membros de equipe colaborando em tempo real, integrador de API externa.
