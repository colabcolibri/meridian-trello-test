# v1 — MVP kanban local

## Objective

A v1 entrega o Trello minimalista prometido no escopo: aplicativo desktop que abre direto no workspace, permite criar quadros e colunas editáveis, gerenciar cartões ricos (etiquetas, prioridade, prazo, checklist, observações) e mover tarefas por drag and drop — tudo persistido localmente sem login ou internet. O foco é utilidade imediata para um único usuário no próprio computador.

## Done criteria

A v1 está completa quando um gestor ou usuário final consegue clonar o repositório, seguir o README, abrir o app em desenvolvimento ou instalar o build, criar um quadro com colunas padrão editáveis, popular cartões com metadados, mover cards entre colunas via DnD, fechar e reabrir o app com dados intactos, e todos os Must US estiverem ✅ com evidência no Record.

## Included in this version

- EPIC-01 — Fundação Tauri, SQLite e camada de dados.
- EPIC-02 — Quadros e navegação inicial do workspace.
- EPIC-03 — Colunas kanban editáveis e ordenáveis.
- EPIC-04 — Cartões: criação, compacto, mover, arquivar e duplicar.
- EPIC-05 — Modal de detalhes, etiquetas, checklist, vencimento e observações.
- EPIC-06 — Drag and drop e layout kanban responsivo.
- EPIC-07 — README, scripts de dev/build e entrega utilizável.

## Explicitly out

- Contas, sync na nuvem e colaboração multiusuário — produto diferente; v2+ se houver demanda.
- Mobile app nativo — v1 limita-se a layout responsivo desktop/notebook.
- Temas customizáveis e atalhos avançados — polish pós-MVP.
