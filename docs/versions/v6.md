# v6 — Demo web in-browser (sem backend)

## Objective

O produto hoje só roda no Tauri; quem abre o link do repo não experimenta o fluxo projeto → versão → épico → sprint → US. A v6 entrega um **modo demo no navegador**: mesma UI React, persistência **apenas** em `localStorage` no dispositivo do visitante, sem API nem SQLite remoto. Banner deixa claro que nada sobe para servidor.

## Done criteria

Gestor valida: `pnpm build` + abrir `dist` no browser (ou Pages) — criar projeto, versão, épico, sprint, US, mover cartão; reload mantém dados no mesmo browser; outro browser/dispositivo não vê os dados. CI publica GitHub Pages na branch `agile`. US v6 ✅ com Record.
