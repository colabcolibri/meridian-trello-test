# US-0007 — Layout colunas com scroll horizontal

**As** Usuário local,
**I want** ver colunas lado a lado com rolagem horizontal,
**so that** eu visualize todo o fluxo kanban em telas menores.

## Intent

### Acceptance

- [x] Colunas renderizadas em row flex dentro do quadro ativo.
- [x] Scroll horizontal na faixa de colunas; body da página sem overflow-x indesejado.
- [x] Largura mínima de coluna ~280px; layout utilizável em 1024px+.

### Why

Layout kanban é identidade do produto. Antes desta US, quadro aberto não mostra pipeline. Depois, estrutura visual existe para receber CRUD de colunas e cartões.

### Where

v1-S3 / EPIC-03. Depende de US-0006. Desbloqueia US-0008 e US-0009.

## Plan

### Approach

- Criar `src/features/boards/ui/BoardView.tsx` como container do quadro ativo, carregando metadados do board via `boardService` e renderizando `ColumnStrip` — separar layout de dados de colunas (US-0008).
- Implementar `ColumnStrip` em `src/features/columns/ui/ColumnStrip.tsx` com `display: flex`, `overflow-x: auto`, `min-width: 280px` por coluna e estilos em `src/app/styles/kanban.css`, conforme `ui-kanban.md`.
- Permitir colunas mock ou lista vazia com skeleton até US-0008; foco desta US é scroll e contenção de overflow no body.
