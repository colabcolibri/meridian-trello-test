# US-0029 — Modal de cartão visual polish

**As** Usuário local,
**I want** um modal de cartão limpo e profissional,
**so that** editar detalhes seja confortável por vários minutos.

## Intent

### Acceptance

- [x] Modal branco `max-w-[768px]` com sombra e cantos arredondados — não bloco cinza inteiro.
- [x] Header com list name + título proeminente + botão fechar discreto.
- [x] Sidebar cinza clara com campos e ações em estilo botão Trello (`.trello-modal-field`, `.trello-modal-action`).
- [x] Área principal com description/checklist/notes espaçadas.
