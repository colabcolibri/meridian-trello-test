# US-0024 — Cartão compacto e overlay de arraste estilo Trello

**As** Usuário local,
**I want** cartões legíveis e arraste com feedback visual polido,
**so that** eu escaneie tarefas rapidamente e mova cards com confiança.

## Intent

### Acceptance

- [x] `CardCompact` com sombra suave, hover elevado, padding e tipografia alinhados a Trello.
- [x] Labels (tags) como faixas coloridas no topo do card quando presentes.
- [x] Badges de due date, prioridade e checklist discretos no rodapé do card.
- [x] `DragOverlay` com rotação/sombra Trello-like; placeholder tracejado na origem durante drag.
- [x] Coluna destino destaca drop zone durante hover (ring ou fundo sutil).

### Why

Cartões são a unidade de trabalho mais vista. O compacto v1 funciona mas parece formulário web; combinado com bug de clone (US-0021), o arraste precisa feedback visual claro para o usuário entender o que está acontecendo.

### Where

v2-S1 / EPIC-09. Depende de US-0021 (DnD estável) e US-0022 (tokens). Integra com colunas de US-0023.

## Plan

### Approach

- Refatorado `CardCompact.tsx` com classes `.trello-card`, faixas `.trello-label-strip` no topo, badges de prazo/checklist no rodapé e prop `variant="overlay"` para o `DragOverlay`.
- `SortableCard` em `BoardView.tsx` renderiza `.trello-drag-placeholder` durante drag; `DragOverlay` usa `CardCompact variant="overlay"`.
- Coluna recebe classe `.is-drop-target` via `useDroppable` quando `isOver` durante arraste.
