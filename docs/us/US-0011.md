# US-0011 — Mover e reordenar cartões pré-DnD

**As** Usuário local,
**I want** mover e ordenar cartões antes do drag and drop,
**so that** o fluxo funcione mesmo enquanto DnD não está pronto.

## Intent

### Acceptance

- [x] Menu ou controle move cartão para outra coluna do mesmo quadro.
- [x] Controles reordenam cartão dentro da coluna (▲▼ ou similar).
- [x] `move_card` / `reorder_cards` persistem; reload reflete ordem.

### Why
