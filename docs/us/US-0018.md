# US-0018 — Drag and drop de cartões

**As** Usuário local,
**I want** arrastar cartões entre colunas,
**so that** eu atualize o andamento com gesto natural de kanban.

## Intent

### Acceptance

- [x] Drag and drop dentro da mesma coluna reordena cartões.
- [x] Drag para outra coluna move cartão e persiste column_id + order.
- [x] Feedback visual durante arraste; rollback se persist falhar.

### Why

DnD é experiência definida na spec — diferencial vs menus de US-0011. Reutiliza commands move/reorder já validados.

### Where

v1-S6 / EPIC-06. Depende de US-0011. Desbloqueia US-0020 (README demo fluxo completo).

## Plan

### Approach

- Instalar `@dnd-kit/core`, `@dnd-kit/sortable`, `@dnd-kit/utilities`; envolver `ColumnStrip` com `DndContext` e cada coluna como `SortableContext` — reutilizar `CardCompact` como draggable item.
- Handler `onDragEnd`: se `over` é mesma coluna, chamar `cardService.reorderCards`; se coluna diferente, `moveCard` com índice destino; optimistic update no state local com rollback se invoke falhar.
- Manter menu ▲▼ de US-0011 como fallback acessível — DnD não remove controles existentes (progressive enhancement).
