# US-0060 — CRUD épico com vínculo relacional à versão

**As** Usuário local,
**I want** vincular épico a versões existentes ao criar ou editar,
**so that** a relação seja real no banco e alimente o Included derivado da versão.

## Intent

### Acceptance

- [ ] `CreateEpicDialog` após create chama `set_epic_versions` com ids selecionados no checkbox picker — não envia `versionIdsJson` string.
- [ ] Novo `EditEpicDialog` carrega épico + versões vinculadas via `list_epics_for_version` inverso ou command `get_epic_versions`; permite alterar capability, outcome, status e vínculos de versão.
- [ ] `ProductCrudPanel` abre EditEpicDialog por épico; remove dependência de persistir `version_ids_json` no blur inline.
- [ ] Ao vincular épico a versão v5-test, `get_version_included` na versão (US-0059) reflete o épico sem ação manual na versão.
- [ ] Épico sem versão selecionada é rejeitado na validação do wizard (mesma regra de hoje).

### Why

O checkbox de versões no create de épico já existe visualmente, mas grava JSON — não participa do Included derivado de forma confiável. Este slice conecta o picker à API relacional de US-0058 e adiciona edição completa do épico, hoje limitada a título/status inline.

### Where

v5-S2 / EPIC-19. Depende US-0058. Complementa US-0059 (Included derivado). Unblocks US-0062 (hub lista épicos por versão). Não altera sprint ou US forms.

## Plan

### Approach

- Atualizar `CreateEpicDialog.tsx`: após `epicService.create`, chamar `epicService.setVersions(projectId, epicId, versionIds)` — novo método thin wrapper sobre invoke `set_epic_versions`; remover `versionIdsJson` do payload create.
- Criar `EditEpicDialog.tsx` reutilizando steps e validação de create; load inicial via `epicService.get` + `epicService.listVersionIds(projectId, epicId)` se necessário.
- Ajustar `epicService.ts` e `src-tauri/src/commands/agile/epics.rs` update path para não sobrescrever `version_ids_json`; junction é SoT.
- `ProductCrudPanel` tab epics: card row com Editar abrindo dialog; manter consistência visual com versão (US-0059).
