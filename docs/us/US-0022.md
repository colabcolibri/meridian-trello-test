# US-0022 — Design tokens e fundo do board estilo Trello

**As** Usuário local,
**I want** abrir um quadro com visual profissional estilo Trello,
**so that** eu me sinta usando uma ferramenta madura, não um protótipo.

## Intent

### Acceptance

- [x] Tokens centralizados (cores de board, texto, sombra, raio) em CSS ou módulo TS reutilizável.
- [x] Área do quadro usa fundo estilo Trello (gradiente ou imagem sutil + overlay) — não `bg-slate-100` genérico.
- [x] Tipografia legível com hierarquia clara (título board, títulos de coluna, corpo de card).
- [x] Contraste WCAG AA mínimo para texto principal sobre fundo do board.

### Why

Hoje toda a UI usa utilitários Tailwind genéricos sem identidade. O usuário local espera experiência Trello desde o escopo; o fundo do board é a primeira impressão ao abrir um quadro e ancora o restante do redesign.

### Where

v2-S1 / EPIC-09. Depende de v1 entregue (US-0020 README). Desbloqueia US-0023, US-0024 e US-0025 que aplicam os mesmos tokens.

## Plan

### Approach

- Criado `src/app/trello-theme.css` com variáveis CSS customizadas (`--trello-*`) cobrindo cores de board, colunas, cards, texto, sombras e raios de borda; o arquivo é importado em `styles.css` para disponibilizar os tokens globalmente em toda a aplicação.
- Aplicado gradiente de fundo ao board (`--trello-board-gradient`) substituindo a classe Tailwind genérica `bg-slate-100`, com hierarquia tipográfica clara definida pelos tokens para título do board, cabeçalhos de coluna e corpo de cartão; contraste validado como WCAG AA.
