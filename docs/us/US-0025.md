# US-0025 — Modal de detalhe e lista de quadros estilo Trello

**As** Usuário local,
**I want** modal de cartão e lista de quadros com visual coerente estilo Trello,
**so that** toda a jornada — do workspace ao detalhe da tarefa — pareça um produto único.

## Intent

### Acceptance

- [x] Modal: layout two-column ou equivalente (conteúdo principal + sidebar metadados), overlay escurecido, tipografia e espaçamento Trello-like.
- [x] Campos existentes (descrição, tags, checklist, prazo, notas, ações) reorganizados sem perder funcionalidade v1.
- [x] Lista de quadros: cards de board com hover, CTA “Criar quadro” proeminente, não lista HTML crua.
- [x] Navegação workspace → board → modal mantém tokens de US-0022.
- [x] Responsivo: modal utilizável em 1024px sem overflow horizontal.

### Why

O modal v1 e a lista de quadros foram implementados como utilitários funcionais. Fechar a v2 exige consistência visual em todas as superfícies — usuário que confia no board também precisa confiar no detalhe da tarefa e na escolha de quadros.

### Where

v2-S1 / EPIC-09. Depende de US-0022 (tokens). Fecha EPIC-09 e sprint v2-S1 junto com US-0023/0024.

## Plan

### Approach

- Refatorado `CardDetailModal.tsx` para layout grid two-column: conteúdo principal (descrição, checklist com barra de progresso) + sidebar cinza com metadados e ações — overlay `.trello-modal-overlay`.
- Refatorado `BoardListPage.tsx` para grid de tiles com capa gradiente por quadro, hover actions e botão “Criar novo quadro” estilo Trello workspace.
- Reutilizados tokens `--trello-*` e classes `.trello-workspace`, `.trello-board-tile`, `.trello-btn-primary` em ambas as superfícies.
