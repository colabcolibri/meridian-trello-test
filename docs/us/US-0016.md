## Intent

### Acceptance

- [x] Modal permite definir/limpar data de vencimento (date picker ou input date).
- [x] Cartão com due_date no passado exibe indicador vermelho no compacto e no modal.
- [x] Cartões sem prazo não mostram badge de vencimento.

### Why
