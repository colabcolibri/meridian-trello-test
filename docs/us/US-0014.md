# US-0014 — Etiquetas coloridas por quadro

**As** Usuário local,
**I want** etiquetas coloridas nos cartões,
**so that** eu categorize visualmente tarefas do mesmo quadro.

## Intent

### Acceptance

- [x] Criar tag com nome e cor no contexto do quadro.
- [x] Modal permite associar/desassociar tags ao cartão.
- [x] Compacto exibe chips com cor da tag.

### Why

Tags são diferencial visual no kanban. Escopo por board evita poluição global — alinhado ao data model.

### Where

EPIC-05, v1-S5. Depende de US-0013. Alimenta US-0010 compacto retroactivamente.

## Plan

### Approach

- Commands `list_tags`, `create_tag`, `set_card_tags` em `src-tauri/src/commands/tags.rs`; service `src/features/tags/tagService.ts` — tags escopadas por `board_id`.
- Componente `TagPicker` no modal reutilizando paleta fixa em `src/domain/constants.ts` (`TAG_COLORS`); `TagChip` compartilhado entre modal e `CardCompact`.
- Ao salvar tags, refetch `useCardDetail` e invalidar coluna pai para compacto atualizar chips coloridos.
